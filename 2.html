input tag

- 입력받고자 하는 형태를 정의
- text, radio, button, checkbox, email, password, url, tel,
file, date, month, week, time, color, number, range, submit...

name

- 각, 입력요소를 구분하는 중요한 속성
- 데이터를 서버에 전달할 때, 변수의 이름으로 사용
- 반드시, 정의해주는 게 좋습니다.

주요 공통 속성

  1. `readonly` : 읽기 전용
  2. `disabled` : 비활성화
  3. 'autocomplete` : 자동 완성
  4. 'autofocus` : 웹 페이지 로드 시 초기에 포커싱 설정
  5. 'placeholder` : 입력 폼에 희미하게 설명을 보여줌
  6. 'required` : 필수 속성
  7. 'spellcheck` : 오타 점검
